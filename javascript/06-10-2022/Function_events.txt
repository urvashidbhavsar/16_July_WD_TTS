(1) built-in functions - which is already created in code.
ex, write(), writeln(), focus(), isNan()

(2) user define functions - which is created by us.
syntax :

    statements

what is event?
something performed.
there is 4 types - mouse, keyboard, form, window.
ex, mouseover, keypress, submit, click, load

event Handler?

ex, onclick, onkeypress, onload, onsubmit

(3) prompt - taken input from user
